---

## 1. Propósito del Engagement

Este engagement existe para diseñar y entregar una plataforma de apoyo a la toma de decisiones para la construcción y optimización de portafolios en múltiples mercados financieros.

A partir de las conversaciones iniciales y el análisis realizado, el desafío central no es el acceso a datos, sino la capacidad de:

- Evaluar de forma objetiva el trade-off entre riesgo y retorno
- Comparar configuraciones de portafolios con rigor matemático
- Reducir la incertidumbre en decisiones de inversión multi-mercado

La plataforma no está pensada como una herramienta de trading.
Es un sistema de análisis estratégico, basado en la Teoría Moderna de Portafolios, diseñado para mejorar la calidad de las decisiones.

---

## 2. Definición de Éxito

Este engagement se considera exitoso cuando el cliente cuenta con:

- Una aplicación web lista para producción que:
  - Genere portafolios optimizados usando modelos cuantitativos
  - Visualice claramente riesgo, retorno y trade-offs
  - Permita comparar alternativas de inversión de forma objetiva
- Una arquitectura preparada para:
  - Escalar a nuevos mercados e instrumentos
  - Funcionar como base para futuras estrategias de monetización

El éxito se mide por claridad, confiabilidad y extensibilidad, no por cantidad de funcionalidades.

## 4. Capacidades del Sistema (Alcance Orientado a Resultados)

Lo siguiente describe qué habilita el sistema, independientemente de cómo se implemente técnicamente.

### 4.1 Optimización de Portafolios

- Optimización media-varianza (framework de Markowitz)
- Restricciones definidas por el usuario
- Generación de frontera eficiente
- Métricas clave:
  - Retorno esperado
  - Volatilidad
  - Ratio de Sharpe
  - Indicadores de riesgo a la baja

### 4.2 Análisis Histórico y de Riesgo

- Evaluación histórica multi-anual
- Simulación de escenarios de portafolios
- Análisis de drawdown
- Comparaciones entre escenarios ponderados

### 4.3 Supuestos Personalizables

- Ajuste manual de:
  - Retornos esperados
  - Volatilidades
  - Correlaciones
- Cálculos automáticos por defecto
- Persistencia de supuestos por usuario

### 4.4 Microfinanzas (Caso de Uso Definido)

- Análisis con datos trimestrales
- Optimización bajo restricciones mínimas de retorno
- Comparación de rendimiento acumulado

### 4.5 Interfaz Orientada a Decisión

- Dashboard web interactivo
- Visualizaciones claras y accionables:
  - Riesgo vs retorno
  - Composición del portafolio
  - Evolución histórica
- Diseñada para analistas y tomadores de decisión, no traders

## 5. Inclusiones y Exclusiones

### Incluido

- Desarrollo end-to-end de la plataforma web
- Motor de optimización de portafolios
- Persistencia de datos
- Autenticación de un usuario
- Despliegue inicial en producción

### No Incluido

- Aplicaciones móviles
- Trading en tiempo real
- Integraciones con brokers
- Asesoramiento financiero o regulatorio
- Soporte o mantenimiento continuo post-entrega
